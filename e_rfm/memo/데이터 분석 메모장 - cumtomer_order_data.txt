※ 가중치 곱하기 ※
구한 R, F, M 각 값에 임의의 가중치를 산정해서 곱하여 가중치를 높게 설정한 데이터가 더 잘 분석되게 하자. 가중치의 값은 모두 더해서 1이 되게 잘 조절해서 3개의 값에 가중치를 설정하자.

데이터를 분석하는 플랫폼이....
- 회원이 자주 접속하는 것을 중점으로 본다? 					-> R의 가중치를 가장 높게.
- 회원이 자주 활동(거래, 구매등...)하는 것을 중점으로 본다? 			-> F의 가중치를 가장 높게.
- 회원이 많이 구매하는 것(빈도가 아닌 금액수준)을 중점으로 본다?		-> M의 가중치를 가장 높게.

0.3, 0.5, 0.2 -> 이런식으로 분배하고....
* 1.3, * 1.5, *1.2 -> 이런식으로 곱해서 가중치를 설정한다.

★ 구한 RFM 값중 R에는 1을 빼서 값을 재구성하자!

정규화 수치값을 구하면 ndarray가 되기에, 해당 배열을 다시 데이터프레임을 만들고,
구한 RFM 값을 행마다 모두 합하면 Total_Score 값이 된다.
해당 값으로 기존 데이터들과 관계를 분석해볼 수 있다!
